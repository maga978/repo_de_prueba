## Getting Started

# Lógica de Programación y Proyectos en Java 🚀

¡Bienvenido/a a mi repositorio de aprendizaje y desarrollo en Java! Este espacio está dedicado a consolidar mis conocimientos en los fundamentos del lenguaje, la resolución de problemas lógicos y la creación de proyectos interactivos estructurados.

El repositorio está organizado de manera evolutiva: comenzando con ejercicios clásicos de consola y avanzando hacia aplicaciones y juegos más complejos utilizando estructuras de datos.

## 🛠️ Tecnologías y Herramientas Utilizadas
* **Lenguaje:** Java (JDK 17 o superior)
* **Entorno de Desarrollo:** Visual Studio Code (con la suite de extensiones para Java)
* **Control de Versiones:** Git y GitHub Desktop
